# 💰 Finance Helper Chatbot

An AI-powered finance chatbot built using **Streamlit** and **Groq API**.
This chatbot helps users with basic financial guidance such as budgeting, saving, investing, and managing expenses.

---

## 🚀 Features

* 💬 Interactive chat interface
* 💰 Financial advice (budgeting, saving, investing)
* ⚡ Fast responses using Groq LLM
* 🧠 Context-aware conversations
* 🎯 Beginner-friendly financial explanations

---

## 🛠️ Tech Stack

* **Python**
* **Streamlit**
* **Groq API (LLM)**
* **python-dotenv**

---

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/finance-chatbot.git
cd finance-chatbot
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

Or manually:

```bash
pip install streamlit groq python-dotenv
```

---

## 🔑 Setup Environment Variables

Create a `.env` file in the root directory:

```
GROQ_API_KEY=your_api_key_here
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

---

## 🌐 Access the App

Open your browser and go to:

```
http://localhost:8501
```
